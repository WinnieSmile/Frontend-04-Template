# 发布系统
发布系统分为了三个子系统：
* 线上服务系统：给真正的用户提供线上服务用的
* 发布系统：程序员向线上服务系统去发布所用的所谓的发布系统，它可以跟线上服务系统是同级部署，也可以跟线上服务系统是两个独立的集群。
单机、同级部署发布系统和线上服务器。
* 发布工具：与发布系统相连接，需要用一个命令行工具去跟它对接。
